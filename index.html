PolyP - a polytypic programming language

Patrik Jansson and Johan Jeuring

PolyP version 1.0

This distribution contains
  • the source files for a compiler written in Haskell that compiles PolyP-code to Haskell.
  • a library of polytypic functions written in PolyP
  • some examples. (For more info see the README file.)
The compiler is written in Haskell 1.4 (+ the ST-monad) and is known to work with
  • Chalmers Haskell compiler hbc 0.9999.4,
  • Hugs 1.4 January 1998 and
  • Glasgow Haskell compiler 2.10.

Download

Download the source code, unpack with
gunzip polyp.tar.gz
tar xf polyp.tar
and follow the installation instructions. Please send a mail to patrikj@cs.chalmers.se if you download it, or have comments or bugs to report.
