PolyP - a polytypic programming language

Patrik Jansson, Johan Jeuring and Ulf Norell

PolyP version 2

This distribution contains
  • the source files for a compiler written in Haskell that compiles PolyP-code to Haskell.
  • a library of polytypic functions written in PolyP
  • some examples. (For more info see the examples/README file and the draft reference manual.)
The compiler is written in Haskell 98 (+ the ST-monad) and is known to work with
  • Glasgow Haskell (version 5.04.3, 6.0.1) and
  • Hugs 98 (Dec 2001).

Download

Download the source code, unpack with
gunzip polyp.tar.gz
tar xf polyp.tar
and follow the installation instructions. Please send a mail to NOpatrikjSP@csA.chalmersM.se if you download it, or have comments or bugs to report.
